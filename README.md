# FinanceApp

A personal/business financial operating system for Android.

## Features
- **Dashboard** – Real‑time summary of income, expenses, buffer, and runway
- **Transactions** – Add, edit, and delete transactions with dynamic category/subcategory filtering
- **Receipts** – Generate and share PDF receipts for any transaction
- **Data Management** – Delete transactions by date range, export full financial statements as PDF
- **Offline‑first** – All data stored locally with Room (SQLite)

## Tech Stack
- Language: Kotlin
- UI: XML layouts with Material Components
- Database: Room
- Architecture: MVVM (ViewModel + StateFlow)

## Setup
- Clone the repo
- Open in Android Studio or VS Code with Android tools
- Build and run on a device/emulator (min SDK 26)
